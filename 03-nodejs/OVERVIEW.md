# Stage 3 — Node.js Engineering

## Purpose

Build robust backend, CLI, and automation systems using Node.js.

## Topics

- Filesystem and paths
- Processes and environment variables
- Streams and buffers
- Events and child processes
- Fetch, HTTP, retries, timeouts
- Logging, config, error handling
- Packaging and scripts

## Drills

- Build retry wrappers and timeout helpers
- Implement file walkers and stream pipelines
- Wrap external APIs with robust error behavior

## Mini-projects

1. Multi-command CLI utility
2. Resilient API integration service
3. Local automation script suite

## Exit criteria

You can ship a reliable Node tool or service with sane logging, config, and failure handling.
