# Stage 5 — Agent Core Systems

## Purpose

Understand how to build agentic systems as software systems rather than prompt demos.

## Topics

- Observation → reasoning → action loops
- Tool abstraction and registries
- State and memory boundaries
- Planning vs direct execution
- Logging, traces, and debugging
- Safeguards and failure recovery
- Evaluation basics

## Drills

- Build a manual agent loop
- Add a tool execution layer
- Track agent state across turns
- Emit logs for each step and decision

## Mini-projects

1. Research helper agent
2. Multi-step local workflow agent
3. Tool-using assistant with trace logs

## Exit criteria

You can explain and implement a small agent architecture with tools, state, and observability.
