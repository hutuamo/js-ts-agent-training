# Stage 7 — Production Hardening

## Purpose

Turn working prototypes into systems you can trust, demo, and operate.

## Topics

- Validation and contracts
- Retries, timeouts, backoff, fallbacks
- Logging and metrics
- Prompt/version management
- Testing and evaluation harnesses
- Security and secrets hygiene
- Deployment options and operational basics

## Drills

- Add structured logs and request IDs
- Build regression tests for prompts/tools
- Add rate-limit handling
- Add defensive validation at every external boundary

## Mini-projects

1. Harden an existing agent project
2. Add an eval suite and benchmark cases
3. Deploy a minimal service or bot

## Exit criteria

You can explain how your agent behaves under failure and prove basic reliability with logs, tests, and evals.
