# Stage 1 — JavaScript Foundations

## Purpose

Learn enough JavaScript to read, write, debug, and modify medium-sized codebases confidently.

## Topics

- Values, variables, coercion
- Functions, closures, scope
- Objects, arrays, iteration
- Prototype basics
- Modules
- Errors and exceptions
- JSON, dates, regex
- Async basics: Promise and async/await

## Drills

- Rewrite classic C-style utilities in JS
- Parse structured text into JSON
- Build simple CLIs
- Read and explain existing JS code

## Mini-projects

1. File summarizer CLI
2. Weather/API client CLI
3. Batch JSON transformer

## Exit criteria

You can build small tools without tutorial-following and are comfortable reading common JS patterns.
