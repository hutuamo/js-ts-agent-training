# Stage 0 — Foundations, Environment, and Mindset

## Purpose

Establish the working environment and mental model for transitioning from static compiled languages into modern JavaScript/TypeScript and AI application engineering.

## Topics

- Node.js, npm/pnpm, package.json, npx
- JavaScript runtime vs compiled language expectations
- Event loop and asynchronous thinking
- Modules, tooling, debuggers, REPL
- C/C++ to JS/TS mental model translation
- Development journal and learning workflow

## Exercises

1. Install Node LTS and package manager of choice.
2. Create a hello-world CLI and run it via `node`.
3. Configure a formatter/linter.
4. Write a short note: “What assumptions from C/C++ will fail in JS/TS?”

## Exit criteria

You can set up a project, run scripts, inspect package metadata, and explain the biggest runtime/model differences between C++ and JS.
