# Stage 8 — Capstone Projects

## Purpose

Demonstrate end-to-end capability with portfolio-grade projects.

## Capstone options

### 1. Research Assistant
- ingest PDFs/web pages/notes
- summarize and extract tasks/questions
- maintain citation-backed outputs

### 2. Codebase Analyst
- inspect a repository
- summarize architecture
- answer structural questions
- generate review notes or refactor suggestions

### 3. Personal Operations Agent
- interpret user tasks
- route to tools and workflows
- generate plans, reports, and follow-ups

## Required artifacts

- README
- architecture note
- setup guide
- limitations
- demo script or screenshots
- roadmap for future improvements

## Exit criteria

You can publish and explain a serious JS/TS AI agent project with confidence.
