# Stage 6 — Retrieval, Memory, and Workflows

## Purpose

Learn when to use retrieval and workflows instead of hoping a free-form agent will solve everything.

## Topics

- Document ingestion and chunking
- Embeddings and retrieval basics
- Citation and source grounding
- Short-term vs long-term memory
- Workflow composition and orchestration
- Human-in-the-loop boundaries

## Drills

- Build a local retrieval prototype
- Add citations to answers
- Compare workflow solutions vs autonomous agents
- Persist summaries or memory safely

## Mini-projects

1. Document QA assistant
2. Research notebook generator
3. Retrieval-backed workflow runner

## Exit criteria

You can build a grounded, retrieval-aware assistant and justify when a workflow is better than a fully agentic loop.
