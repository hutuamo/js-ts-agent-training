# JS/TS for AI Agent Engineering Training

A complete staged training system for learning JavaScript, TypeScript, Node.js, and AI Agent engineering from zero-to-advanced, inspired by Microsoft-style progressive Rust training tracks.

## Goal

Train an experienced systems programmer (for example with C/C++ background) to become productive in **JavaScript/TypeScript AI Agent development** through a structured path:

- **Stage 0**: Setup & mindset
- **Stage 1**: JavaScript foundations
- **Stage 2**: TypeScript foundations
- **Stage 3**: Node.js engineering
- **Stage 4**: LLM and tool-calling basics
- **Stage 5**: Agent systems design
- **Stage 6**: Retrieval, memory, workflows
- **Stage 7**: Production hardening
- **Stage 8**: Capstone projects

This repository is designed as a **training system**, not just a reading list.

## Who this is for

- You already know how to program, but are new to JavaScript/TypeScript.
- You want to build AI agents, not just read about them.
- You prefer a structured path with milestones, drills, projects, and progression gates.

## Repository Structure

- `00-foundations/` — environment, mindset, how JS/TS differs from C/C++
- `01-javascript/` — JS core language and runtime thinking
- `02-typescript/` — TS essentials and practical type design
- `03-nodejs/` — Node engineering, async I/O, packaging, services
- `04-llm-basics/` — model APIs, prompts, structured output, tool calling
- `05-agent-core/` — loops, tools, state, planning, evaluation
- `06-rag-memory-workflows/` — retrieval, memory, workflow systems
- `07-production/` — reliability, observability, testing, deployment
- `08-capstones/` — portfolio-grade end-to-end projects
- `resources/` — reading lists, templates, rubrics, roadmap

## How to use this training

Recommended flow:

1. Read the `OVERVIEW.md` in each stage.
2. Complete the exercises in order.
3. Build the mini-project(s) for that stage.
4. Pass the stage exit criteria before moving on.
5. Keep a development journal and project log.

## Recommended duration

- Fast track: **10-12 weeks**
- Normal track: **3-5 months**
- Deep track: **6+ months**

## Core principles

1. **Build while learning.** No stage is reading-only.
2. **Prefer CLI/backend work first.** Frontend is optional.
3. **Understand the runtime.** Especially async, I/O, modules, and failure modes.
4. **Write types for clarity, not type-golf.**
5. **Agents are systems.** Prompting alone is not enough.
6. **Evaluation matters.** A demo that sometimes works is not mastery.

## Suggested book stack

Primary books:

- JavaScript: *JavaScript: The Definitive Guide (7th ed.)*
- TypeScript: *Programming TypeScript*
- TypeScript quality: *Effective TypeScript*
- Node engineering: *Node.js Design Patterns*

## Deliverables by the end

You should be able to:

- Build a TypeScript project from scratch
- Call model APIs and implement tool-calling loops
- Design a multi-tool agent with memory/state
- Build retrieval-enhanced workflows
- Add logging, retries, validation, and basic evals
- Ship one or more portfolio-grade agent systems
