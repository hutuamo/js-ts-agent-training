# Stage 2 — TypeScript Foundations

## Purpose

Use TypeScript as a practical engineering layer over JavaScript.

## Topics

- Type annotations and inference
- Interfaces and type aliases
- Unions and intersections
- Narrowing and guards
- Generics
- Modules and declaration files
- tsconfig and strict mode
- Runtime validation and schema alignment

## Drills

- Add types to existing JS projects
- Design input/output types for tools
- Model API responses safely
- Refactor weakly-typed code to clearer shapes

## Mini-projects

1. Typed task manager
2. Typed API client
3. Tool registry with typed execute/input/output contracts

## Exit criteria

You can design clear types for application boundaries and avoid the most common TypeScript pitfalls.
