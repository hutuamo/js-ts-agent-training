# 进阶评估标准

## 阶段完成检查

每完成一个阶段前，请确认以下几点：

- [ ] 我完成了该阶段要求的练习。
- [ ] 我完成了该阶段的 mini-project。
- [ ] 我能用自己的话解释核心概念。
- [ ] 我能调试该阶段常见失败场景。
- [ ] 我写下了自己困惑的点，以及我是怎么解决的。

## 能力等级

### Beginner
- 能安全地跟着例子改代码。
- 理解基本语法和执行流程。
- 能做出小型可运行程序。

### Intermediate
- 能独立设计模块。
- 能调试不熟悉的代码。
- 能接入 API 和外部系统。
- 能组织非平凡项目结构。

### Advanced
- 能有意识地做架构取舍。
- 能构建可靠的 workflow 和 agent。
- 能加入校验、日志和评估。
- 能解释系统行为和限制。

### Expert 方向
- 能指导别人并批判性 review 设计。
- 能构建可复用的内部框架或库。
- 能处理运维、安全和权限约束。
- 能系统性地评估和提升 agent 可靠性。
