# Stage 4 — LLM and Tool-Calling Basics

## Purpose

Move from ordinary TypeScript backend work into AI application development.

## Topics

- Model API fundamentals
- Prompt roles and message formats
- Structured output
- Streaming responses
- Tool / function calling
- Cost, latency, and failure awareness
- Configuration and secrets handling

## Drills

- Build a single-turn assistant CLI
- Extract structured JSON from text
- Implement one tool-calling roundtrip
- Add retries and validation to model output

## Mini-projects

1. CLI chat assistant
2. Structured extraction tool
3. Typed tool-calling assistant

## Exit criteria

You can independently integrate an LLM API and build a reliable tool-calling flow.
